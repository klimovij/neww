# 🎨 Настройки фронтенда

## 🎯 Описание

Модалка **"Фронтенд"** позволяет полностью настроить внешний вид приложения:
- ✅ Основной фон (цвет + изображение)
- ✅ Фон модальных окон
- ✅ Стили кнопок сайдбара
- ✅ Цвет, размер и шрифт текста
- ✅ **Живое превью** всех изменений

---

## 🚀 Как открыть

### Десктопная версия:
1. Откройте **"Управление"** (кнопка в сайдбаре или значок шестеренки)
2. Нажмите кнопку **"Фронтенд"** (синяя кнопка с иконкой монитора)

### Мобильная версия:
1. Откройте меню **"Управление"**
2. Нажмите кнопку **"Фронтенд"**

> ⚠️ **Доступ:** Только для администраторов (role: admin)

---

## ⚙️ Доступные настройки

### 1️⃣ Основной фон

**Цвет фона:**
- Выбор через палитру цветов
- Ручной ввод HEX кода (например: `#111827`)

**Изображение фона:**
- 📁 Кнопка **"Выбрать изображение"** - загрузка с компьютера
- Поддерживаемые форматы: JPG, PNG, GIF, WEBP
- Максимальный размер: 5 МБ
- 🗑️ Кнопка **"Удалить изображение"** - удаление фона

**Прозрачность изображения:**
- Ползунок от 0% до 100%
- Позволяет смешивать изображение с цветом фона

---

### 2️⃣ Фон модальных окон

**Цвет фона модалок:**
- Выбор через палитру цветов
- Ручной ввод HEX кода (например: `#1f2937`)
- Применяется ко всем модальным окнам в приложении

---

### 3️⃣ Стили сайдбара

**Фон кнопок:**
- Ручной ввод CSS значения
- Поддерживает rgba, hex, gradient
- Пример: `rgba(59, 130, 246, 0.1)`

**Цвет текста:**
- Выбор через палитру цветов
- Ручной ввод HEX кода
- Применяется ко всему тексту в сайдбаре

**Размер текста:**
- Ползунок от 10px до 20px
- Ручной ввод точного значения
- По умолчанию: 14px

**Шрифт:**
- Выпадающий список с популярными шрифтами:
  - Inter (по умолчанию)
  - Arial
  - Helvetica
  - Segoe UI
  - Roboto
  - Open Sans
  - Lato
  - Georgia
  - Times New Roman
  - Courier New

## 💾 Сохранение настроек

### Кнопки управления:

**Сбросить:**
- Возвращает все настройки к значениям по умолчанию
- Требует подтверждения

**Отмена:**
- Закрывает модалку без сохранения
- Все изменения будут отменены

**Сохранить:**
- Сохраняет настройки в `localStorage`
- Применяет изменения ко всему приложению
- Кнопка активна только если есть изменения
- После сохранения показывается уведомление ✅

## 🔄 Применение настроек

### Автоматическое применение:

Настройки применяются автоматически при:
- ✅ Открытии приложения (загрузка из `localStorage`)
- ✅ Сохранении в модалке настроек
- ✅ Синхронизации между вкладками браузера

### CSS переменные:

Все настройки сохраняются как CSS переменные:
```css

--main-background-image
--main-background-image-opacity
--modal-background
--sidebar-button-background
--sidebar-text-color
--sidebar-text-size
--sidebar-font-family
```

## 🔧 Технические детали

### Хранение данных:
- **localStorage:** `frontendSettings`
- **Формат:** JSON
- **Размер:** ~5-10 KB (без изображения), до 7 MB (с изображением Base64)

### Изображения:
- **Формат хранения:** Base64
- **Преимущества:** Не требует сервера, работает офлайн
- **Недостатки:** Занимает больше места в localStorage

### Синхронизация:
- **Между вкладками:** Автоматически через `storage` event
- **Между устройствами:** Пока не реализовано (localStorage локален)

---

## ⚠️ Ограничения

1. **Размер изображения:** Максимум 5 МБ
2. **Формат изображения:** Только JPG, PNG, GIF, WEBP
3. **localStorage:** ~10 MB на домен (зависит от браузера)
4. **Синхронизация:** Только в рамках одного браузера

---

## 💡 Советы

### Для лучшего результата:

1. **Контрастность:** Убедитесь, что текст читаем на выбранном фоне
2. **Размер изображения:** Используйте оптимизированные изображения (< 1 MB)
3. **Прозрачность:** 20-40% для изображений обычно выглядит лучше всего
4. **Шрифты:** Для кириллицы лучше использовать Inter, Roboto или Arial
5. **Тестирование:** Проверьте как выглядит в разных разделах приложения

### Если что-то пошло не так:

1. Нажмите кнопку **"Сбросить"**
2. Очистите localStorage: `F12` → Console → `localStorage.removeItem('frontendSettings')` → `F5`
3. Перезагрузите страницу

## 🆘 Проблемы и решения

### Проблема: Изображение не загружается
**Решение:** 
- Проверьте размер файла (< 5 MB)
- Проверьте формат (JPG, PNG, GIF, WEBP)
- Попробуйте другое изображение

### Проблема: Настройки не сохраняются
**Решение:**
- Проверьте, что нажали кнопку "Сохранить"
- Проверьте консоль браузера (F12) на ошибки
- Очистите кеш браузера

### Проблема: Текст не читается
**Решение:**
- Измените цвет текста на контрастный
- Уменьшите прозрачность фонового изображения
- Выберите более темный/светлый фон модалок
