# Быстрая проверка после обновления

## Сервер обновлён ✅

Frontend пересобран, PM2 перезапущен. Теперь нужно проверить работу.

## Проверка на сервере

Выполните на сервере для проверки API:

```bash
# Проверка, что username есть в ответе
curl 'http://localhost/api/quick-db-report?start=2025-11-24&end=2025-11-24' | jq '.report[0] | {username, fio}'

# Должно вывести что-то вроде:
# {
#   "username": "Ksendzik_Oleg",
#   "fio": "Олег Ксендзик"
# }

# Проверка данных активности
curl 'http://localhost/api/activity-summary?start=2025-11-24&end=2025-11-24' | jq '.summary[] | select(.username=="Ksendzik_Oleg")'

# Должно вывести данные активности
```

## Проверка в браузере

1. Откройте веб-интерфейс: http://35.232.108.72
2. Войдите в систему
3. Откройте модалку "Work Time Report" (или "Рабочее время")
4. Выберите сегодняшнюю дату (2025-11-24)
5. Нажмите "Показать отчет"
6. Нажмите кнопку "Детали" (или "Показать детали") для любого пользователя
7. Откройте консоль браузера (F12 → Console)
8. Должны появиться логи:
   ```
   🔍 [WorkTimeReportModal] Загружаем activity-summary...
   🔍 [WorkTimeReportModal] Row data: {username: "Ksendzik_Oleg", fio: "Олег Ксендзик"}
   🔍 [WorkTimeReportModal] Ищем activityStats для username: Ksendzik_Oleg
   ✅ [WorkTimeReportModal] Найденный activityStats: {...}
   ```
9. В модалке должны отображаться:
   - **Активность за период**
   - Количество активных минут
   - Количество минут простоя
   - Топ приложений

## Если данные не отображаются

1. **Проверьте логи сервера:**
   ```bash
   sudo -u appuser pm2 logs mesendger-server --lines 100 | grep -i activity
   ```

2. **Проверьте консоль браузера** - должны быть логи с префиксом `🔍`

3. **Очистите кэш браузера** (Ctrl+F5) и обновите страницу

4. **Проверьте, что username совпадает:**
   - В консоли браузера должно быть: `Row data: {username: "Ksendzik_Oleg", ...}`
   - И: `Доступные usernames в summary: ["Ksendzik_Oleg"]`
   - Если они не совпадают - проблема в данных

## Если всё ещё не работает

Пришлите:
1. Вывод команды `curl` с проверкой API
2. Скриншот консоли браузера с логами
3. Скриншот модалки (если возможно)
