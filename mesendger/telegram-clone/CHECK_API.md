# Проверка API applications

## Выполните на сервере:

1. **Откройте мониторинг логов в реальном времени:**
```bash
sudo -u appuser pm2 logs mesendger-server --lines 0
```

2. **В другом окне/терминале выполните прямой запрос к API:**
```bash
curl "http://localhost:3000/api/activity-details?username=Ksendzik_Oleg&start=2025-11-25&end=2025-11-25" | jq '.applications | length'
```

3. **Или откройте в браузере детали пользователя и смотрите логи в реальном времени**

## Что искать в логах:

- `🔍 [activity-details] Всего логов для` - сколько всего логов
- `🔍 [activity-details] Уникальных proc_name` - сколько уникальных процессов
- `📊 [activity-details] Приложений для пользователя` - сколько приложений найдено
- `⚠️ [activity-details] НЕТ ПРИЛОЖЕНИЙ!` - предупреждение, если нет приложений

## Если приложений нет:

Проверьте, что фильтры не исключают все записи. В логах будет видно, какие proc_name отфильтрованы.
