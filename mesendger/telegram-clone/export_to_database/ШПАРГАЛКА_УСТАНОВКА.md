## ✅ Скрипты универсальные - ничего менять НЕ НУЖНО!

---

## 🚀 Установка (3 минуты на 1 ПК):

### 1. Подготовка (один раз):
- Скопируйте папку `export_to_database` на флешку или сетевой диск

### 2. На каждом ПК:

```powershell
# 1. Откройте PowerShell от имени администратора

# 2. Перейдите в папку (замените E:\ на букву флешки)
cd E:\export_to_database

# 3. Запустите установку
.\setup_pc_monitoring.ps1

# 4. Введите username сотрудника (например: Ivanov_Ivan)
# 5. Нажмите Enter 3 раза (интервалы по умолчанию: 5 минут)
# 6. Готово! ✅
```

---

## 📋 Что устанавливается:

| Что | Где |
|-----|-----|
| **Скрипты** | `C:\pc-worktime\` |
| **Настройки** | `%APPDATA%\mesendger\agent_config.json` |
| **Логи** | `%APPDATA%\mesendger\*.log` |
| **Задачи** | Планировщик задач Windows |

---

## ✅ Проверка:

```powershell
# Проверить задачи
schtasks /Query /TN Mesendger_PC_Startup_Monitor
schtasks /Query /TN Mesendger_PC_Shutdown_Monitor
schtasks /Query /TN Mesendger_PC_Activity_Monitor

# Проверить файлы
dir C:\pc-worktime
dir $env:APPDATA\mesendger

# Проверить логи
notepad $env:APPDATA\mesendger\pc_startup.log
```

---

## ⚠️ Важно:

1. **Username** должен совпадать с базой Mesendger
2. **Требуются права администратора**
3. **Сервер:** `http://35.232.108.72` (одинаковый для всех ПК)
4. **Пути:** одинаковые на всех ПК, ничего менять не нужно

## 🔧 Если не работает:

```powershell

# Проверить сеть

# Исправить username
```
