# ⚡ БЫСТРАЯ УСТАНОВКА ЭКСПОРТА УДАЛЕНКИ

## 🎯 Что это?
Автоматическая отправка данных входа/выхода с удаленного сервера (где работают 40 человек) на Google сервер каждый день в 7:00.

---

## 📦 БЫСТРАЯ УСТАНОВКА (5 минут)

### 1️⃣ Скопируйте 3 файла на удаленный сервер:
```
📁 export_to_database/
   ├── export_remote_worktime.ps1
   ├── setup_remote_worktime_export.ps1
   └── Mesendger_Remote_Worktime_Export.xml
```

### 2️⃣ PowerShell **от Администратора**:
```powershell
cd "C:\путь\к\mesendger\telegram-clone\export_to_database"
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser -Force
.\setup_remote_worktime_export.ps1
```

### 3️⃣ Готово! ✅
Задача создана и будет запускаться каждый день в 7:00.

## ⚠️ ПРОБЛЕМЫ? (90% случаев)

### ❌ "Database file not found"
```powershell
# Проверить путь к базе:
dir "..\server\messenger.db"
dir "..\..\server\messenger.db"
```

### ❌ "Node.js not found"
```powershell
# Установить Node.js:
# https://nodejs.org/ → Download → Install → Перезапустить PowerShell
node --version  # должна показаться версия
```

### ❌ "No events found for date"
```powershell
# База пустая - проверить приложение:
# 1. Запущено ли приложение Mesendger?
# 2. Работает ли система логирования?
# 3. Есть ли данные в базе?
```

### ❌ Задача не запускается автоматически
```
Планировщик → Задача → Свойства → Общие:
✅ "Выполнять с наивысшими правами" - ДА
✅ Пользователь: текущий пользователь
✅ Выполнять вне зависимости от входа - НЕТ

Триггеры:
✅ Время: 07:00:00
✅ Повтор: Ежедневно
✅ Включено: Да
```

## 📊 ГДЕ СМОТРЕТЬ РЕЗУЛЬТАТЫ?

### 1. На удаленном сервере (логи):
```powershell
# Информация о задаче
Get-ScheduledTask -TaskName Mesendger_Remote_Worktime_Export | Get-ScheduledTaskInfo

# Журнал задачи
taskschd.msc → Задача → Журнал (вкладка внизу)
```

### 2. На Google сервере (данные):
```
Сайт: http://35.232.108.72
Меню → Мониторинг времени → Отчет Удаленка
Выбрать: Дата + Пользователь
✅ Должны отображаться входы/выходы
```

## 💡 ПОЛЕЗНЫЕ КОМАНДЫ

```powershell
# Отправить данные за последние 7 дней
for ($i = 1; $i -le 7; $i++) {
    $date = (Get-Date).AddDays(-$i).ToString("yyyy-MM-dd")
    Write-Host "📅 $date" -ForegroundColor Cyan
    .\export_remote_worktime.ps1 -Date $date
    Start-Sleep -Seconds 2
}

# Проверить содержимое базы
cd "..\server"
npm install -g sqlite3
sqlite3 messenger.db "SELECT COUNT(*) FROM work_time_logs WHERE date(event_time) = '2025-11-30';"

# Проверить сетевое подключение
Test-NetConnection -ComputerName 35.232.108.72 -Port 5000

# Посмотреть следующий запуск задачи
(Get-ScheduledTask -TaskName Mesendger_Remote_Worktime_Export | Get-ScheduledTaskInfo).NextRunTime
```
