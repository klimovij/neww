# Мониторинг включения/выключения ПК

## Описание

Скрипты для автоматического мониторинга включения и выключения ПК с отправкой данных на сервер Mesendger.

## Файлы

- `send_pc_startup.ps1` - отправляет событие login (включение ПК)
- `send_pc_shutdown.ps1` - отправляет событие logout (выключение ПК)
- `install_pc_monitoring_tasks.ps1` - устанавливает задачи в планировщике Windows

## Установка

### 1. Ручная установка

1. Откройте PowerShell **от имени администратора**
2. Перейдите в папку со скриптами:
   ```powershell
   cd C:\Users\Ronin\web\pc-worktime
   ```
3. Запустите скрипт установки:
   ```powershell
   .\install_pc_monitoring_tasks.ps1
   ```

### 2. Что делает скрипт установки

- Создает задачу `Mesendger_PC_Startup_Monitor` - запускается при включении ПК
- Создает задачу `Mesendger_PC_Shutdown_Monitor` - запускается при выключении ПК
- Скрипты выполняются в скрытом режиме (без окон)
- Задачи запускаются от имени SYSTEM с максимальными правами

### 3. Проверка установки

Проверьте созданные задачи:
```powershell
schtasks /Query /TN Mesendger_PC_Startup_Monitor
schtasks /Query /TN Mesendger_PC_Shutdown_Monitor
```

### 4. Тестирование

**Тест включения:**
```powershell
.\send_pc_startup.ps1
```

**Тест выключения:**
```powershell
.\send_pc_shutdown.ps1
```

## Удаление задач

Если нужно удалить задачи:
```powershell
schtasks /Delete /TN Mesendger_PC_Startup_Monitor /F
schtasks /Delete /TN Mesendger_PC_Shutdown_Monitor /F
```

## Важные замечания

### Включение ПК
- ✅ Работает надежно через триггер "При запуске системы"
- Скрипт запускается автоматически при каждой загрузке Windows

### Выключение ПК
- ⚠️ Windows не поддерживает прямой триггер "при выключении"
- Используется триггер на событие Event Log (Event ID 1074, 6008)
- Если триггер события не работает, создается альтернативная задача "при входе пользователя"
- Для более надежной работы рекомендуется использовать групповую политику

## Альтернативный способ для выключения

Если задача с триггером события не работает, можно использовать групповую политику:

1. Откройте `gpedit.msc` (Групповая политика)
2. Перейдите: `Конфигурация компьютера` → `Конфигурация Windows` → `Сценарии (запуск/завершение работы)`
3. В разделе "Сценарии завершения работы" добавьте скрипт `send_pc_shutdown.ps1`

## Просмотр данных

После установки данные будут автоматически отправляться на сервер и отображаться в модалке:
- **Мониторинг времени** → **Отчет активности локальных ПК**
- При нажатии "Подробнее" → вкладка "События" - все включения и выключения ПК

## Логи

Логи скриптов сохраняются в:
- `%APPDATA%\mesendger\pc_startup.log` - для включения
- `%APPDATA%\mesendger\pc_shutdown.log` - для выключения

## Конфигурация

Username пользователя сохраняется в:
- `%APPDATA%\mesendger\agent_config.json`

При первом запуске скрипт запросит username, который будет сохранен для последующих запусков.
