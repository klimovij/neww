# Отображение событий выключения в модалке "Подробности"

## ✅ Изменения внесены!

События выключения ПК теперь отображаются в модалке "Подробности" вместе с событиями входа/выхода.

## Что изменилось

### 1. Скрипт выключения (`send_pc_shutdown_with_fallback.ps1`)
- ✅ Теперь отправляет `event_type: "shutdown"` вместо `"logout"`
- ✅ Использует `event_id: 1074` (Windows Event ID для выключения системы)
- ✅ Это позволяет отличать выключение ПК от обычного выхода пользователя

### 2. Клиент (`UserWorkTimeDetailsMobile.jsx`)
- ✅ Добавлена поддержка отображения событий типа `shutdown`
- ✅ События выключения отображаются с **оранжевым цветом** (#ff9800)
- ✅ Используется иконка **FiPower** (иконка питания)
- ✅ Текст: **"Выключение"**

### 3. Статистика
- ✅ Добавлен счетчик **"Выключений"** в разделе статистики
- ✅ Отображается оранжевым цветом

### 4. Сервер (`report.js`)
- ✅ Добавлена обработка `event_type: "shutdown"`
- ✅ Автоматически присваивается `event_id: 1074` для событий выключения

## Визуальное отображение

### События в модалке "Подробности":

1. **Вход** (зеленый) - `event_type: "login"`
   - Иконка: FiLogIn
   - Цвет: #43e97b (зеленый)

2. **Выход** (красный) - `event_type: "logout"`
   - Иконка: FiLogOut
   - Цвет: #e74c3c (красный)

3. **Выключение** (оранжевый) - `event_type: "shutdown"` ⭐ НОВОЕ
   - Иконка: FiPower
   - Цвет: #ff9800 (оранжевый)

### Статистика:

```
Всего событий: X
Входов: Y (зеленый)
Выходов: Z (красный)
Выключений: W (оранжевый) ⭐ НОВОЕ
```

## Как это работает

1. При выключении ПК запускается `send_pc_shutdown_with_fallback.ps1`
2. Скрипт отправляет на сервер:
   ```json
   {
     "username": "Ksendzik_Oleg",
     "event_type": "shutdown",
     "event_time": "2025-11-30 20:20:14",
     "event_id": 1074
   }
   ```
3. Сервер сохраняет событие в базу данных
4. При открытии модалки "Подробности" событие отображается с оранжевым цветом и иконкой питания

## Проверка

После установки и тестирования:

1. Выключите ПК
2. Включите ПК
3. Откройте модалку "Подробности" для пользователя
4. Вы должны увидеть событие "Выключение" с оранжевым цветом
5. В статистике должен появиться счетчик "Выключений"
