# Интеграция нового скрипта выключения

## ✅ Интеграция завершена!

Новый скрипт `send_pc_shutdown_with_fallback.ps1` **полностью интегрирован** в существующую систему установки через `setup_pc_monitoring.ps1`.

## Как это работает

### Автоматическая установка (рекомендуется)

Просто запустите существующий скрипт установки:

```powershell
.\setup_pc_monitoring.ps1
```

Скрипт автоматически:
1. ✅ Скопирует новый скрипт `send_pc_shutdown_with_fallback.ps1` в `C:\pc-worktime`
2. ✅ Обновит XML файл `Mesendger_PC_Shutdown_Monitor.xml` для использования нового скрипта
3. ✅ Создаст задачу в планировщике задач, которая будет использовать новый скрипт
4. ✅ Обновит `send_pc_startup.ps1` для автоматической отправки сохраненных данных

### Что изменилось

#### 1. XML файл (`Mesendger_PC_Shutdown_Monitor.xml`)
- ✅ Обновлен для использования `send_pc_shutdown_with_fallback.ps1` вместо `send_pc_shutdown.ps1`
- ✅ Использует более надежные Event ID: 1074, 1076, 6008 (вместо 12, 13)
- ✅ Увеличен таймаут до 5 секунд (вместо 3)

#### 2. Скрипт установки (`setup_pc_monitoring.ps1`)
- ✅ Добавлено копирование нового скрипта `send_pc_shutdown_with_fallback.ps1`
- ✅ Старый скрипт `send_pc_shutdown.ps1` также копируется для совместимости

#### 3. Скрипт запуска (`send_pc_startup.ps1`)
- ✅ Добавлена функция `Send-PendingShutdown` для проверки и отправки сохраненных данных
- ✅ При каждом запуске автоматически проверяет наличие файла `pending_shutdown.json`
- ✅ Автоматически отправляет сохраненные данные о выключении

## Преимущества новой системы

### До (старая система)
- ❌ Если интернет недоступен при выключении - данные теряются
- ❌ Если сервер недоступен - данные теряются
- ❌ Если ПК выключается слишком быстро - данные могут не успеть отправиться

### После (новая система)
- ✅ Данные сохраняются в файл при любой ошибке отправки
- ✅ Данные автоматически отправляются при следующем запуске ПК
- ✅ Надежная работа даже при быстром выключении
- ✅ Полная совместимость с существующей системой

## Файлы

### Основные файлы
- `send_pc_shutdown_with_fallback.ps1` - новый скрипт с резервным сохранением
- `send_pc_startup.ps1` - обновленный скрипт запуска (проверяет сохраненные данные)
- `Mesendger_PC_Shutdown_Monitor.xml` - обновленный XML для планировщика задач

### Вспомогательные файлы
- `setup_shutdown_task.ps1` - отдельный скрипт для установки только задачи выключения (если нужно)
- `SHUTDOWN_TASK_README.md` - подробная документация

## Использование

### Вариант 1: Полная установка (рекомендуется)
```powershell
.\setup_pc_monitoring.ps1
```
Установит все компоненты, включая новый скрипт выключения.

### Вариант 2: Только задача выключения
```powershell
.\setup_shutdown_task.ps1 [username]
```
Установит только задачу выключения с новым скриптом.

## Проверка работы

### Проверка задачи
```powershell
schtasks /Query /TN Mesendger_PC_Shutdown_Monitor
```

### Проверка логов
- Лог выключения: `%APPDATA%\mesendger\pc_shutdown.log`
- Лог запуска: `%APPDATA%\mesendger\pc_startup.log`
- Отложенные данные: `%APPDATA%\mesendger\pending_shutdown.json`

### Ручной тест
```powershell
.\send_pc_shutdown_with_fallback.ps1 "Ksendzik_Oleg"
```
