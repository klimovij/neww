# 📅 ЭКСПОРТ ДАННЫХ ЗА ПЕРИОД

## 🎯 Назначение

Скрипт `export_remote_worktime_range.ps1` позволяет отправить данные за несколько дней **одной командой**.

---

## 🚀 БЫСТРОЕ ИСПОЛЬЗОВАНИЕ

### Пример 1: За весь ноябрь (с 1 по 28 число)

```powershell
.\export_remote_worktime_range.ps1 -StartDate 2025-11-01 -EndDate 2025-11-28
```

### Пример 2: За последнюю неделю

```powershell
.\export_remote_worktime_range.ps1 -StartDate 2025-11-22 -EndDate 2025-11-28
```

### Пример 3: Формат DD.MM.YYYY

```powershell
.\export_remote_worktime_range.ps1 -StartDate 01.11.2025 -EndDate 28.11.2025
```

### Пример 4: С увеличенной задержкой (5 секунд между запросами)

```powershell
.\export_remote_worktime_range.ps1 -StartDate 2025-11-01 -EndDate 2025-11-28 -DelaySeconds 5
```

---

## 📋 ПАРАМЕТРЫ

| Параметр | Обязательный | Описание | Пример |
|----------|--------------|----------|--------|
| `-StartDate` | ✅ Да | Начальная дата | `2025-11-01` или `01.11.2025` |
| `-EndDate` | ✅ Да | Конечная дата | `2025-11-28` или `28.11.2025` |
| `-DelaySeconds` | ❌ Нет | Задержка между запросами (по умолчанию: 2 сек) | `5` |

## ⚠️ ВАЖНЫЕ МОМЕНТЫ

### 1. Задержка между запросами

**По умолчанию: 2 секунды**

Зачем нужна задержка?
- ✅ Не перегружает сервер
- ✅ Дает время на обработку
- ✅ Избегает rate limiting

**Когда увеличить:**
- 🐌 Медленный интернет → `-DelaySeconds 5`
- 🌐 Много данных → `-DelaySeconds 10`
- ⚡ Быстрая сеть → `-DelaySeconds 1`

### 2. Время выполнения

**Формула:**
```
Время ≈ Количество дней × (2 секунды задержки + 1 секунда обработки)
```

**Примеры:**
- 7 дней ≈ 21 секунда (~0.4 минуты)
- 28 дней ≈ 84 секунды (~1.4 минуты)
- 30 дней ≈ 90 секунд (~1.5 минуты)

### 3. Дни без данных

Это нормально для:
- 📅 Выходных (суббота, воскресенье)
- 🎉 Праздников (когда офис не работает)
- 🔧 Дней до установки системы логирования

### 4. Ошибки

Частые причины:
- ❌ Нет интернета
- ❌ Сервер недоступен
- ❌ API ключ неверный
- ❌ База данных заблокирована

**Решение:** Скрипт покажет дни с ошибками, можно запустить повторно только для них.

## 🆘 РЕШЕНИЕ ПРОБЛЕМ

### Проблема 1: "Скрипт не найден: export_remote_worktime.ps1"

**Причина:** Файлы в разных папках

**Решение:**
```powershell
# Убедитесь, что оба файла в одной папке:
dir export_remote_worktime.ps1
dir export_remote_worktime_range.ps1
```

### Проблема 2: "Неверный формат даты"

**Причина:** Неправильный формат

**Решение:** Используйте один из форматов:
```powershell
# Формат 1: YYYY-MM-DD
-StartDate 2025-11-01

# Формат 2: DD.MM.YYYY
-StartDate 01.11.2025
```

### Проблема 3: Много ошибок подряд

**Причина:** Проблемы с сервером или сетью

**Решение:**
1. Проверьте интернет:
```powershell
Test-NetConnection -ComputerName 35.232.108.72 -Port 80
```

2. Проверьте сервер:
```powershell
Invoke-WebRequest -Uri "http://35.232.108.72" -Method GET
```

3. Увеличьте задержку:
```powershell
-DelaySeconds 10
```

### Проблема 4: "Начальная дата позже конечной"

**Причина:** Даты перепутаны местами

**Решение:** Поменяйте `StartDate` и `EndDate` местами:
```powershell
# ❌ Неправильно:
-StartDate 2025-11-28 -EndDate 2025-11-01

# ✅ Правильно:
-StartDate 2025-11-01 -EndDate 2025-11-28
```
