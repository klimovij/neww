ПРОБЛЕМА:
  ✅ Задача PC_Activity_Agent существует
  ❌ Процесс агента не запущен
  ❌ Задача не запускается автоматически при входе в систему

РЕШЕНИЕ (2 минуты):

1. Откройте PowerShell от имени АДМИНИСТРАТОРА:
   - Нажмите Win + X
   - Выберите "Windows PowerShell (администратор)" или "Терминал (администратор)"

2. Перейдите в папку:
   cd "C:\Users\Ronin\web\pc-worktime"

3. Запустите скрипт установки:
   .\setup_pc_worktime_tasks.ps1

4. Скрипт автоматически:
   ✅ Удалит старую задачу (если она зависла)
   ✅ Создаст новую задачу с правильными настройками
   ✅ Настроит автозапуск при входе в систему
   ✅ Добавит перезапуск при сбое

5. Проверьте результат - должно появиться:
   ✅ Создана/обновлена задача: PC_Activity_Agent
   ✅ Задача включена и готова к запуску

6. ПРОВЕРКА (необязательно, но рекомендуется):
   - Откройте Планировщик задач (Win + R → taskschd.msc)
   - Найдите задачу PC_Activity_Agent
   - Проверьте статус: должно быть "Готово" (Ready)
   - Проверьте триггер: должен быть "При входе в систему" (At log on)

7. ПЕРЕЗАГРУЗИТЕ ПК для проверки автозапуска:
   - После входа в систему подождите 1-2 минуты
   - Откройте Диспетчер задач (Ctrl + Shift + Esc)
   - Вкладка "Процессы" → найдите "pwsh.exe" или "powershell.exe"
   - Проверьте папку C:\pc-worktime\logs\ → должны появиться новые файлы
