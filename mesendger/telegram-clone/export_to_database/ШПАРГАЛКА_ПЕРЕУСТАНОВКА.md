# ⚡ Переустановка на ПК со старыми версиями - Шпаргалка

## Ситуация:
У пользователей **УЖЕ установлены** старые скрипты/задачи, которые работают некорректно.  
Нужно **удалить старые** и **установить новые**.

---

## 🎯 Вариант 1: Автоматическая переустановка (РЕКОМЕНДУЕТСЯ)

### Один скрипт делает ВСЁ:

```powershell
# 1. PowerShell от имени администратора

# 2. Перейти в папку
cd E:\export_to_database  # E:\ = флешка

# 3. Запустить переустановку
.\ПЕРЕУСТАНОВИТЬ.ps1

# Скрипт автоматически:
# - Удалит все старые задачи ✅
# - Использует сохраненный username (не нужно вводить заново) ✅
# - Установит новые задачи ✅
```

**Время: ~2 минуты на 1 ПК**

---

## 🎯 Вариант 2: Ручная переустановка (2 шага)

### Шаг 1: Удалить старые задачи

```powershell
# PowerShell от имени администратора
cd E:\export_to_database
.\УДАЛИТЬ_СТАРЫЕ_ЗАДАЧИ.ps1

# Скрипт спросит:
# - Удалить файлы из C:\pc-worktime? → N (Нет, оставить)
# - Удалить настройки %APPDATA%\mesendger? → N (Нет, сохранить username)
```

### Шаг 2: Установить новую версию

```powershell
.\setup_pc_monitoring.ps1

# Username уже сохранен → просто Enter × 3 для интервалов
```

**Время: ~3 минуты на 1 ПК**

---

## 🎯 Вариант 3: Только удаление (без установки)

Если нужно только удалить старые задачи:

```powershell
.\УДАЛИТЬ_СТАРЫЕ_ЗАДАЧИ.ps1

# Выберите:
# [Y] - удалить всё (файлы, настройки, логи)
# [N] - удалить только задачи
```

---

## 📋 Что удаляется:

| Задача | Что |
|--------|-----|
| ✅ Автоматически | Все задачи в планировщике |
| ⚠️ По выбору | Файлы `C:\pc-worktime\` |
| ⚠️ По выбору | Настройки `%APPDATA%\mesendger\` |

**Рекомендация:**
- Задачи → **удалить** (автоматически)
- Файлы → **оставить** (будут перезаписаны новыми)
- Настройки → **оставить** (чтобы не вводить username заново)

---

## ✅ Проверка после переустановки:

```powershell
# Проверить задачи
schtasks /Query /TN Mesendger_PC_Startup_Monitor
schtasks /Query /TN Mesendger_PC_Shutdown_Monitor
schtasks /Query /TN Mesendger_PC_Activity_Monitor

# Проверить файлы
dir C:\pc-worktime

# Проверить username
notepad $env:APPDATA\mesendger\agent_config.json
```

---

## 🔧 Список удаляемых задач:

- ✅ `Mesendger_PC_Startup_Monitor`
- ✅ `Mesendger_PC_Shutdown_Monitor`
- ✅ `Mesendger_PC_Activity_Monitor`
- ✅ `Mesendger_PC_Logoff_Monitor` (если есть)
- ✅ `Mesendger_PC_Shutdown_Monitor_Background` (если есть)

---

## ⚠️ Важно:

1. **Требуются права администратора** для всех операций
2. **Username сохраняется** - не нужно вводить заново (если выбрали "N" при удалении настроек)
3. **Старые логи** можно оставить - они не мешают работе
4. **Перезагрузка не требуется** - работает сразу

## 📞 Если что-то пошло не так:

### Задачи не удаляются:
```powershell
# Проверьте права администратора
net session
```

### Установка не запускается:
```powershell
# Разрешите выполнение скриптов
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
```

### Username потерян:
```powershell
# Посмотрите в базе Mesendger или запустите setup_pc_monitoring.ps1 вручную
```

3. 🟢 `setup_pc_monitoring.ps1` - только установка
